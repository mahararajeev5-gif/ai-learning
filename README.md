## 🔥 Featured Project

### 🖐️ Hand Gesture AI with Voice

An AI-based system that detects hand gestures and responds with Hindi voice.

### 🚀 Features

* Real-time hand tracking using webcam
* Finger detection using MediaPipe
* Hindi voice output (Text-to-Speech)
* Multiple gesture support

### 🖐️ Gestures

* 1 Finger → Ram Ram 🙏
* 2 Fingers → Hello 👋
* 3 Fingers → Love You ❤️

---

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe
* gTTS / pyttsx3
